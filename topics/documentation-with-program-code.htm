Writing Documentation with Program Code

 

 

 

 

HelpSmith allows you to insert and format program code with a syntax highlighting scheme making the code easy-to-read and look similarly to the styles used for a specific programming language or file format.

 

The Syntax Highlighter tool in HelpSmith provides schemes for more than 50 programming languages including most popular languages, such as C++, C#, Visual Basic, Java, PHP, Python, JavaScript, HTML, etc.

 

Thus, you can use HelpSmith to create API documentation, program specifications, help files with code examples, or any other kinds of documentation containing program code written in various programming languages.
